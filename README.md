# Student Management App

Ứng dụng quản lý sinh viên xây dựng bằng Flet, theo đúng hướng tiếp cận từ đề tài và chia theo mô hình OOP.

## Cấu trúc
- app/models.py: các lớp dữ liệu chính
- app/database.py: khởi tạo SQLite và dữ liệu mẫu
- app/services.py: logic nghiệp vụ
- app/ui.py: giao diện Flet
- main.py: điểm khởi chạy

## Chạy ứng dụng
```bash
python main.py
```
